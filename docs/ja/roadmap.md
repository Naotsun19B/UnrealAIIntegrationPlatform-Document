# ロードマップ

以下は現在計画中の機能です。リリース日は未確定であり、内容は変更される場合があります。

---

## 近期予定

### Sandbox Editing（サンドボックス編集）
AI が提案した編集をサンドボックスに仮置きし、人間が内容を確認・承認してからディスクに反映するワークフローを実現します。すべての変更を選択的に受け入れ・破棄でき、Undo に頼る必要がありません。

### アセット監査・依存関係解析
AI エージェントからアセット依存グラフを取得し、未参照アセットの検出・循環参照の特定・コンテンツツリー全体のサイズマップ生成を行います。AI 主導のアセット最適化ループを設計した機能です。

### アセットバリデーション
`UEditorValidatorSubsystem` に登録済みのバリデーターを、単一アセットやフォルダ単位で AI から実行できるようにします。結果は構造化 JSON の Artifact として返却されるため、CI/CD のゲートチェックとして利用できます。

### Asset Manager 設定
PrimaryAssetType 定義・アセットバンドル・アセットタグをプログラマブルに管理します。DLC・コンテンツバンドル・クック対象制御など、`DefaultGame.ini` と Asset Manager に依存するワークフローを対象にしています。

## 中期予定

### ローカライゼーションパイプライン自動化
ローカライゼーションの全工程を自動化します：ソーステキストのギャザー・コンパイル・カルチャー管理・StringTable のエントリ追加/編集/削除・翻訳確認のためのエディタ表示言語切替。AI に翻訳レビューパスを任せたいチームを対象にしています。

### ビルド・パッケージパイプライン
AI コマンドからコンテンツのクック・プロジェクトのパッケージング・Project Launcher プロファイルの実行ができるようになります。長時間処理の進捗報告とキャンセルに対応し、CI/CD パイプラインへの組み込みを想定しています。

### パフォーマンス Insights トレーシング
チャンネル指定で UE Trace セッションを開始・停止し、フレーム統計やヒッチのサマリ、ドメイン別トレース（HTTP イベント・Niagara タイミング・レンダーコマンド）を照会します。結果は JSON Artifact として返却され、AI による解析に利用できます。

### マテリアルバリデーション・テンプレート
マテリアルをプロジェクトルールに照らして検証し、重複防止のために類似マテリアルを検出、ワークフローテンプレートから新規マテリアルを作成します。

## 探索的

### 人間向けエディタ GUI
AI の活動を人間がモニタリングするためのオプショナルエディタタブです。**CommandHistory**（コマンドとレスポンスのタイムライン）・**ArtifactViewer**（スクリーンショット・JSON ダンプ・テストレポートのインラインプレビュー）・**ScenarioBuilder**（JSON を手書きせずにシナリオを組み立てて実行できるビジュアル UI）を想定しています。コアには影響しないオプショナルモジュールとして実装します。

### セマンティックアセット検索
ファイル名ではなく意味でアセットを検索できる AI 駆動のコンテンツブラウザ検索機能です。現時点では埋め込み生成の安定性の問題により開発を一時停止しています。

### 追加グラフエディタ統合

- **Niagara Subsonic**：MetaSound・Niagara グラフへの Subsonic オーディオイベントノード対応（UE 5.8 Subsonic システム）
- **ControlRig Dynamics**：ControlRig グラフ内の簡易物理シミュレーションノード設定
- **Mixed Control Rig トラック**：Level Sequence への Mixed Control Rig トラック追加
- **AnimationLayering / UAF**：Anim Blueprint でのボーンマスクレイヤーと Unified Animation Framework ノード操作
- **MeshPartition（MegaMesh）**：大規模メッシュへの空間分割・非破壊モディファイア設定
- **ChaosCloth Asset**：Weight Map・Sim/Render Mesh・クロス物理シミュレーション設定の意味的コマンド
- **CustomizableSequencerTracks**：Blueprint 定義のカスタム Sequencer トラック型への対応
- **DataPrep Asset**：DataPrep インポートパイプラインアセットの実行・検査

### EDA トランスポート
既存の MCP・HTTP・WebSocket・CLI に加え、Epic の Epic Developer Assistant（EDA）へ接続するオプショナルトランスポートです。
